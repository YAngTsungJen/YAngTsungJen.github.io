  • 在學習dom event的事件處理前，我們先來了解什麼是節點樹狀圖。

瀏覽器是從上到下一行一行的讀取，DOM tree 就是把所有的節點拆開後，彼此的關係會像一棵樹，故取名之，想要透過 DOM API 取得節點，透過 JavaScript 可以這樣寫。
